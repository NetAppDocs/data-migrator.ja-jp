= NetApp Data Migrator の機能、ファイルサーバー、プロトコルのサポート
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
NetApp Data Migratorは、特定の機能、ファイルサーバー、およびプロトコルをサポートしています。



== サポート対象の機能とサポート対象外の機能

NetApp Data Migratorは、NFSおよびSMB移行のためのさまざまな機能をサポートしています。一部の機能はサポートされていません。



=== サポートされる機能

[cols="1,2a"]
|===
| 概要 | サポートされている機能 


| 権限と監査処理  a| 
* 保管連鎖レポート
* 任意アクセス制御リスト（DACL）
* アクセス制御リスト（ACL）を保持する
* ユーザー識別子（UID）とセキュリティ識別子（SID）の再マッピング




| ファイルシステムオブジェクトとリンク  a| 
* ハードリンクの処理
+
NetApp Data Migratorは、オブジェクトとリンクを別々のファイルとして移行します。

* シンボリック リンク
+
*注*：NFSマイグレーションでサポートされています





| ファイルメタデータの保持  a| 
* アクセス時間を維持する
* 作成時刻を保持する
* 変更時刻を保持する
* 権限を保持する




| 移行動作とデータの整合性  a| 
* 削除の伝播
* 段階的移行
* オープンファイルのサポート
* スイッチオーバーのサポート




| 移行の制御と実行  a| 
* ファイルパターンの除外
* 移行のスケジュール設定


|===


=== サポートされない機能

[cols="1,2a"]
|===
| 概要 | サポートされていない機能 


| 権限と監査処理  a| 
システムアクセス制御リスト（SACL）

*注*：SMB移行はサポートされていません



| ファイルシステムオブジェクトとリンク  a| 
* NTFSジャンクションに従う
* シンボリック リンク
+
*注*：SMB移行はサポートされていません

* 代替データストリーム
+
NetApp Data Migratorは代替データストリームを検出できますが、移行することはできません。





| ファイルメタデータの保持  a| 
選択的なファイル属性



| 移行動作とデータの整合性  a| 
スナップショットの移行

|===


== サポートされているファイルサーバ

NetApp Data Migratorは、データ移行のソースおよび移行先として、特定のファイルサーバーをサポートしています。

[cols="1,2a"]
|===
| 概要 | サポートされているファイルサーバー 


| ソース  a| 
任意のNASサーバー（Dell Isilon、ONTAP、Vanilla Linux、Windows、Cloud Volumes ONTAPなど）



| デスティネーション  a| 
Azure NetApp Files（ANF）、Google Cloud NetApp Volumes（GCNV）、Amazon FSx for NetApp ONTAP（FSxN）、Cloud Volumes ONTAPのすべてのサービスレベル

|===


== サポートされているNFSおよびSMB移行プロトコル

NetApp Data Migratorは、データ移行のために特定のNFSおよびSMBプロトコルバージョンをサポートしています。

[cols="1,2a"]
|===
| プロトコル | サポート対象のバージョン 


| NFS  a| 
4.1、3.0



| SMB  a| 
3.1、3.0、2.0

|===
